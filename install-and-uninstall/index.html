犀牛EPOS的安装和卸载

Waloyn
2024-12-21
基础, 教程

请注意，在安装前关闭所有杀毒软件，例如360，否则可能会导致安装失败。

下载

打开犀牛EPOS的官网，下载最新安装包。
1.犀牛EPOS的官网：http://www.xiniuboss.com/
2.下载地址：http://www.xiniuboss.com/index/PCDownload
3.点击下载最新版，下载安装包。
注意：下载时请留意保存路径，Windows一般默认下载到【下载】文件夹内。

安装

1.双击打开下载好的安装程序，开始安装。

2.如果有弹出【用户账户控制】，请点击【是】。

3.选择安装路径，点击【安装】。
注意：安装路径请选择C盘以外的路径，否则可能会导致安装失败，或者安装后无法正常使用。

4.等待安装完成，完成后点击【完成】。

卸载

1.打开【设置】，点击【应用】，接着点击【应用和功能】。

2.应用列表搜索框里输入【xiniu】，找到【Xiniu EPos】。

3.点击【…】，选择【卸载】。

4.在弹出的确认框里点击【卸载】，等待卸载完成。
